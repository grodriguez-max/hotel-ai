Telegram
Ideal para hotel y staff

Telegram está activo cuando el gateway está corriendo y el token está configurado. Los huéspedes envían un mensaje al bot y Aria responde al instante.

Para que un huésped pueda chatear con Aria:
  1. El huésped abre Telegram y busca tu bot
  2. Envía cualquier mensaje — Aria le dará un código
  3. Tú apruebas el código en la sección Pareamiento
  4. Aria empieza a responder normalmente
💬
WhatsApp
El canal preferido de los huéspedes

WhatsApp requiere vincular un número de teléfono escaneando un código QR, similar a WhatsApp Web.

⚠️ Importante
Usa un número dedicado para el hotel, no tu celular personal. Una vez vinculado, solo puede usarse en un dispositivo a la vez.
Para vincular WhatsApp, ejecuta este comando en la terminal del servidor:
openclaw channels login
Luego escanea el QR con tu teléfono → WhatsApp → Dispositivos vinculados → Vincular dispositivo.

  • Cuando cambias las credenciales (API key, token de Telegram)

🔗 Aprobar acceso de Telegram

Cuando alguien le escribe a tu bot de Telegram por primera vez, Aria le pide un código de seguridad. Ese huésped o miembro del staff te comparte ese código y tú lo apruebas aquí.

Cómo funciona:
1
El huésped escribe cualquier mensaje al bot de Telegram
2
Aria le responde con un código de 8 letras/números
3
El huésped te comparte ese código
4
Tú lo ingresas aquí y haces clic en "Aprobar"

📧 Conexión con Gmail
1. Activa verificación en 2 pasos en tu cuenta Google.
2. Ve a myaccount.google.com → Seguridad → Contraseñas de aplicaciones y crea una para "Correo".
3. En Gmail activa IMAP: Configuración → Ver toda la configuración → Reenvío e IMAP → Habilitar IMAP.
